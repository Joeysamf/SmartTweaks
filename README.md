# SmartTweaks

A rig mod for DonutSMP. Press O to open the Rigging menu.

## Recent

**Check releases for the latest version of the mod**
https://github.com/Joeysamf/SmartTweaks/releases


## Building

Requires Java 21+.

```bash
./gradlew build
```

The JAR will be in `build/libs/`.

## Usage

1. Place `SmartTweaks-1.21.11.jar` in your `mods/` folder
2. Press **O** in-game
3. Config auto-generated at `config/rigmod.json`
